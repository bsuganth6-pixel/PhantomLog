What PhantomLog Detects

Full transparency on detection logic — no black boxes. All pattern matching happens locally; query strings and paths are URL-decoded first so encoded payloads (e.g. %3Cscript%3E) aren't missed.

SSH Brute-Force (auth.log)

Groups failed logins by source IP using a sliding time window (default: 5+ failures within 5 minutes). If a successful login follows a brute-force burst from the same IP, severity escalates to CRITICAL — possible account compromise.
